# ☕ CoffeeShop Home Page

A cozy **custom browser homepage** built with **React**.  
It recreates the calm atmosphere of a coffee shop while giving you a clean and minimal start page for your browser.

Instead of opening a blank tab, you get a visually pleasant homepage with quick access to your favorite sites.

## 🚀 Getting Started

### 1. Clone the repository

```
git clone https://github.com/mbsurya19122004/CoffeeShopHomePage.git
```

### 2. Go into the project directory

```
cd CoffeeShopHomePage
```

### 3. Install dependencies

```
npm install
```

### 4. Run the development server

```
npm run dev
```

---

## 🏗 Build for Production

```
npm run build
```

This will create an optimized production build inside the **dist/** or **build/** folder.

---

## 🌐 Using It As a Browser Homepage

1. Build the project

```
npm run build
```

2. Serve
   
```
npx serve dist
```
